HTML

생활코딩을 통해 HTML을 공부하며 배운 태그들을 정리해보려 합니다.

HEAD



  1. !doctype html 코드를 가장 위에 작성합니다.

  2. 이후 html 코드를 열고 head 코드를 엽니다. head 코드 하단에 title 코드를 작성합니다.
    head는 실제 웹페이지에는 표시되지 않지만, 웹페이지 생성에 중요한 metadata를 포함합니다.
    title은 웹페이지의 탭에 표시되는 제목을 의미합니다.

  3. meta charset 코드를 작성합니다.
    meta charset는 html 문서에서 반영할 언어 체계를 의미하며 editor의 하단(utf-8 등)에서 확인 가능합니다.

  4. head를 닫습니다.
